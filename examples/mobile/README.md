# Async Storage mobile examples

## Running the example

1. Get dependencies

```bash
$ yarn install
```

2. Install the app

```bash
# Android
$ yarn start:android

# iOS
$ yarn start:ios
```

3. Run packager 

```bash
$ yarn start
```
